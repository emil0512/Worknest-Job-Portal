# 🪹 WorkNest - Job Portal Platform

> **A comprehensive job portal platform with AI-powered resume matching, ATS (Applicant Tracking System), role-based dashboards, and career counseling integration.**

---

## 🎯 About The Project

**WorkNest** is a complete job portal solution designed to connect job seekers, employers, and career counselors on a single platform. It leverages AI-powered resume parsing and intelligent job matching to provide personalized job recommendations.

### Problem It Solves
- Job seekers struggle to find relevant positions
- Employers receive unqualified applications
- No centralized platform for career counseling

### Solution
- Smart resume parsing extracts skills and experience
- AI matching algorithm calculates job fit scores
- Integrated counseling sessions with payment system

---

## ✨ Key Features

### 👥 Multi-Role System

| Role | Capabilities |
|------|--------------|
| **Administrator** | Full platform control, user management, analytics, content moderation |
| **Job Seeker** | Profile management, resume upload, AI job matching, application tracking, saved jobs, counseling booking |
| **Employer** | Job posting, applicant review, ATS shortlisting, interview scheduling, company profile |
| **Career Counselor** | Session scheduling, payment tracking, feedback collection, seeker management |

### 🤖 Intelligent Features

| Feature | Description |
|---------|-------------|
| **Resume Parsing** | Extracts skills, education, experience from PDF resumes |
| **Job Matching Algorithm** | Calculates match scores (0-100%) based on keyword analysis |
| **ATS (Applicant Tracking)** | Auto-shortlisting based on resume-job compatibility |
| **Real-time Messaging** | Direct communication between all user roles |
| **OTP Verification** | Email-based OTP for secure registration |

### 💰 Monetization

- Paid counseling sessions with integrated payment flow
- Transaction tracking for counselors
- Payment history and receipts

### 📊 Analytics

- Admin dashboard with Chart.js visualizations
- Job match score distribution
- User activity tracking
- Application status analytics
